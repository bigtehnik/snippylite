# Floating Text Panel — Плавающая панель для текстовых шаблонов (Windows)

## Обзор

Небольшое приложение на C# (.NET 8, Windows Forms), которое размещает **ультра-компактную плавающую панель** поверх всех окон. Позволяет быстро вставлять текстовые шаблоны в любое активное текстовое поле Windows (браузеры, мессенджеры, редакторы и т.д.).

**Особенности:**
- Панель **20×50 px**, без рамки и иконки в панели задач
- **Левый клик** → меню выбора шаблонов (иерархия групп с бесконечной вложенностью)
- **Правая кнопка**:
  - **Короткий клик** → меню управления (Настройки, Перезагрузить тексты, Выход)
  - **Удержание+движение** → перетаскивание панели по экрану
- Полная поддержка многострочных шаблонов
- Хранение шаблонов в `snippets.json` рядом с exe
- Вставка текста через WinAPI SendInput (Unicode, независимо от раскладки)
- **Окно настроек — древовидный редактор (TreeView)**:
  - Группы отображаются как папки (с иконками)
  - Текстовые команды — как документы
  - Любая глубина вложенности групп
  - Добавление происходит в выбранную группу (или в корень)

## Структура проекта

```
FloatingTextPanel/
├── Program.cs           - точка входа
├── MainPanel.cs         - плавающая панель (Form 20x50)
├── SettingsForm.cs      - окно управления (TreeView)
├── SnippetManager.cs    - менеджер загрузки/сохранения JSON
├── SnippetModels.cs     - модели: MenuNode, Snippet, SnippetCollection
├── TextInserter.cs      - вставка через WinAPI SendInput
├── snippets.json        - хранилище шаблонов (создаётся автоматически)
└── FloatingTextPanel.csproj
```

## Сборка

### Требования
- Visual Studio 2022 с поддержкой .NET 8 Windows Forms
- Windows 10/11

### В Visual Studio
1. Откройте `ws/FloatingTextPanel.sln`
2. Выберите конфигурацию **Release** и платформу **x64**
3. **Сборка → Построить решение**
4. Исполняемый файл: `bin/Release/net8.0-windows/FloatingTextPanel.exe`

### Через CLI
```bash
cd ws/FloatingTextPanel
dotnet publish -c Release -r win-x64 --self-contained false /p:PublishSingleFile=false
```

## Использование

1. Запустите `FloatingTextPanel.exe`.
2. Панель появится в верхнем левом углу (20×50 px, цвет светло-стальной синий, при наведении темнеет).
3. **Перетаскивание**: зажмите **правую кнопку мыши** на панели и двигайте — переместите, куда удобно.
4. **Левый клик** → выпадающее меню со всеми шаблонами (группы → подгруппы → текст). Иконки: папки и документы.
5. **Правый клик** (короткий) → меню:
   - **Настройки...** → открывает окно управления
   - **Перезагрузить тексты** → перечитывает snippets.json
   - **Выход** → закрывает программу

### Окно настроек (дерево)

- Слева — **TreeView**:
  - **Папки** = группы
  - **Документы** = текстовые команды
  - Можно раскрывать/сворачивать
  - Двойной клик по тексту → редактирование
- Справа — свойства выбранного элемента:
  - Для **группы**: только поле «Название»
  - Для **текста**: поля «Название» и «Текст шаблона» (многострочное)
- **Кнопки**:
  - **Добавить** → выбор: «Группа» или «Текст»
    - Если в дереве выбрана группа → новый элемент добавляется внутрь неё
    - Если ничего не выбрано или выбран текст → элемент добавляется в корень
  - **Изменить** → сохраняет изменения имени/текста выбранного элемента
  - **Удалить** → удаляет выбранный элемент (группы удаляются со всем содержимым)
  - **Сохранить** → записывает в snippets.json и закрывает окно
  - **Отмена** → закрывает без сохранения

## Формат snippets.json

Иерархическая модель (без циклических ссылок `Parent`):

```json
{
  "nodes": [
    {
      "name": "Ответы",
      "children": [
        {
          "name": "Поддержка",
          "snippets": [
            { "name": "Спасибо за обращение", "text": "Спасибо за обращение..." }
          ]
        }
      ],
      "snippets": [
        { "name": "Приветствие", "text": "Здравствуйте!" }
      ]
    }
  ],
  "snippets": [
    { "name": "Подпись", "text": "С уважением,\nКоманда" }
  ]
}
```

- `nodes` — дерево групп (может быть произвольная глубина `children`)
- `snippets` — тексты верхнего уровня (не входящие в группы)

При загрузке `Parent` восстанавливаются автоматически.

## Технические детали

### Вставка текста
`TextInserter.InsertText(text)`:
- Активирует активное окно (`GetForegroundWindow` / `SetForegroundWindow`)
- Ждёт 50 мс для стабилизации фокуса
- Отправляет последовательность KEYDOWN/KEYUP для каждого символа через `SendInput` с флагом `KEYEVENTF_UNICODE`
- Не зависит от раскладки, работает с Unicode (кириллица, emoji)

### Перетаскивание панели
Обработка `MouseDown/MouseMove/MouseUp`:
- Запоминаем начальную точку
- При движении превышающем `SystemInformation.DragSize` → начинаем drag: `ReleaseCapture()` + `SendMessage(WM_NCLBUTTONDOWN, HT_CAPTION)`
- Если движение не началось → обрабатываем как клик (ЛКМ/ПКМ)

### Древовидный редактор
- `SettingsForm` работает с копией коллекции (`SnippetCollection GetSnapshot()`)
- При сохранении вызывается `SnippetManager.UpdateCollection(modifiedCollection)`
- Родительские ссылки `Parent` восстанавливаются через `SetParents()` после загрузки и перед сохранением
- `TreeView` использует `ImageList` с иконками: папка (закрытая/открытая) и документ

## Требования к производительности

- Память: < 30 MB
- CPU: ~0% в простое
- Время запуска: < 1 сек

## Ограничения

- Только plain text
- SendInput блокируется UIPI (если целевое окно — админ, а панель — нет)
- Нет автозамены, горячих клавиш, шифрования
- Нет валидации уникальности имён (может быть дубли)
- Панель может уйти за границы экрана при перетаскивании
